# 📚 LLM 학습 도구 - 완벽한 사용 설명서

> **한눈에 보기:** URL 입력 → 본문 추출 → 요약 생성 → 퀴즈 생성

---

## 🎯 이 프로그램은 무엇인가요?

이 프로그램은 **웹페이지의 콘텐츠를 자동으로 분석해서 학습 자료를 만들어주는** AI 도구입니다.

예를 들어:
- 📄 어려운 기사나 블로그 글을 입력하면
- 📝 핵심 내용을 정리해주고
- ❓ 그 내용을 바탕으로 시험 문제(O/X 퀴즈)를 자동으로 만들어줍니다

---

## 📖 단계별 사용법

### 📍 **Step 1: URL 입력하기** (첫 번째 화면)

#### 📝 URL 입력창에 할 일:
1. URL 입력창을 클릭합니다
2. 분석하고 싶은 웹페이지 주소를 입력합니다

#### 💡 URL 예시:
```
https://www.wikipedia.org/wiki/Artificial_intelligence
https://blog.example.com/article/machine-learning
https://en.wikipedia.org/wiki/Climate_change
```

#### ⚠️ 주의사항:
- 반드시 `https://` 또는 `http://`로 시작해야 합니다
- 짧은 URL(예: `example.com`)은 작동하지 않습니다
- 한글 웹사이트도 가능합니다

#### ✅ 올바른 입력:
```
❌ example.com          (안 됨)
❌ www.example.com      (안 됨)
✅ https://example.com  (됨)
```

---

### 🔍 **Step 2: 본문 추출하기**

#### 버튼 위치:
```
[추출] ← 여기를 클릭
[요약]
[퀴즈]
```

#### 무엇이 일어나나요?
1. **URL 입력 후 `[추출]` 버튼 클릭**
2. 웹사이트에서 텍스트를 자동으로 다운로드
3. 메뉴, 광고, 버튼 등 불필요한 부분 제거
4. 순수 본문만 추출 (1-2초 소요)

#### 결과 화면:
```
┌─ 제목: "Artificial Intelligence"
├─ 글자 수: 2,341자
├─ 탭: [본문] [요약] [퀴즈]
└─ 본문 내용:
   "Artificial intelligence (AI) is the simulation..."
   (글 전체가 이곳에 표시됩니다)
```

#### 🎨 모바일에서는:
```
- 글자 수가 위에 표시됩니다
- 탭은 스크롤 가능합니다
- 버튼은 세로로 배열됩니다
```

---

### 📝 **Step 3: 요약 생성하기**

#### 버튼 위치:
```
[추출]
[요약] ← 여기를 클릭
[퀴즈]
```

#### 무엇이 일어나나요?
1. **`[요약]` 버튼 클릭**
2. 추출된 텍스트를 AI(Ollama/llama2)에 전송
3. 핵심 내용을 **한국어로** 2-3 문장으로 요약
4. 결과 표시 (3-10초 소요, 글 길이에 따라 다름)

#### 결과 예시:

**원문 (길고 복잡함):**
> "Artificial intelligence is the simulation of human intelligence processes by machines, especially computer systems. These processes include learning, reasoning, and self-correction. AI has applications in various fields..."

**요약 (짧고 간단함):**
> "인공지능은 기계가 인간의 지능을 모방하는 기술입니다. 학습, 추론, 자체 수정 능력을 포함하며 여러 분야에 응용됩니다."

#### 탭 전환:
```
┌─ [본문] [요약] ← 여기가 파란색으로 변함
└─ 요약 내용:
   "인공지능은 기계가 인간의..."
```

#### ⏱️ 시간이 오래 걸리면:
- Ollama 서버가 처리 중입니다
- 3-10초 정도 기다려주세요
- "처리 중..." 메시지가 보입니다

---

### ❓ **Step 4: 퀴즈 생성하기**

#### 버튼 위치:
```
[추출]
[요약]
[퀴즈] ← 여기를 클릭
```

#### 무엇이 일어나나요?
1. **`[퀴즈]` 버튼 클릭**
2. 본문 내용을 바탕으로 O/X 퀴즈 생성
3. **4-5개의 퀴즈** 자동 생성
4. 각 퀴즈는 난이도/중요도 함께 표시 (5-15초 소요)

#### 퀴즈 화면 구성:

```
┌─ Q1
├─ 난이도: 높음  [높음/중간/낮음 배지]
├─ 중요도: 2/3
├─ 질문: "인공지능은 의료 분야에서 사용되고 있다"
├─ 정답: ⭕ O
└─ 설명: "본문에서 AI의 의료 응용이 명시되어 있습니다"

┌─ Q2
├─ 난이도: 중간
├─ 중요도: 1/3
├─ 질문: "머신러닝은 인공지능의 하위 개념이다"
├─ 정답: ❌ X
└─ 설명: "실제로 머신러닝이 더 광범위한 개념입니다"
```

#### 퀴즈 항목 설명:

| 항목 | 의미 | 예시 |
|------|------|------|
| **Q1, Q2** | 문제 번호 | 첫 번째, 두 번째 문제 |
| **난이도** | 문제 어려움 정도 | 높음(3) / 중간(2) / 낮음(1) |
| **중요도** | 본문에서의 중요도 | 높음 / 중간 / 낮음 |
| **질문** | 참/거짓 판단하는 문장 | "AI는..." |
| **정답** | O(참) 또는 X(거짓) | ⭕ O 또는 ❌ X |
| **설명** | 왜 그 답인지 | "본문의 내용..." |

## ⚠️ 자주 묻는 질문 (FAQ)

### Q1: "에러" 메시지가 나타나요
**A:** 다음을 확인하세요:
- ✅ URL이 `https://`로 시작하나요?
- ✅ 인터넷이 연결되어 있나요?
- ✅ 웹사이트가 접근 불가는 아닌가요?
- ✅ Flask 서버가 실행 중인가요?

### Q2: 요약/퀴즈가 너무 오래 걸려요
**A:** 정상입니다!
- Ollama AI가 생각하는 시간입니다
- 글이 길수록 시간이 더 걸립니다
- 보통 5-15초가 소요됩니다
- 5분 이상 걸리면 브라우저 새로고침 후 다시 시도하세요

### Q3: 퀴즈에 이상한 문자(한자, 일본어)가 섞여있어요
**A:** 이건 AI가 실수한 거예요
- 자동으로 필터링됩니다
- 계속 시도하면 정상 문제가 나옵니다

### Q4: 같은 URL으로 다시 하면 다른 결과가 나와요
**A:** 맞습니다!
- AI가 매번 새로 생성합니다
- 조금씩 다른 요약과 퀴즈가 만들어집니다

### Q5: 모바일에서 버튼이 너무 작아요
**A:** 해결 방법:
- 브라우저에서 "확대" 기능 사용 (Ctrl + / Cmd +)
- 세로 화면 대신 가로 화면으로 보기
- 화면을 더블탭해서 확대하기

### Q6: 요약이 한국어가 아니에요
**A:** 확인 사항:
- ✅ Ollama 서버가 실행 중인가요?
- ✅ 터미널에서 `ollama serve` 실행했나요?
- ✅ 길지 않은 텍스트를 입력했나요? (너무 길면 요약이 안 될 수 있음)
